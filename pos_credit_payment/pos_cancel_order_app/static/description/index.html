Description

Enable POS Cancel Order


Click on Cancel and Reset to Draft to cancel and set the POS order to draft


Click on Cancel Button to change the stage of Order.


Stage change to Draft.


Cancel and Delete.

Select the cancel and delete, and also select cancel delivery to remove the picking of order, user can also select cancel invoice to remove the Invoice of the Order.


POS Order with Invoice and Delivery




Click on Cancel Button.


Order will be Removed with Delivery and Invoice.




Cancel and Draft for Multiple Orders.


All Orders will be set to Draft.


Cancel and Delete.


All the selected orders will be Deleted.
